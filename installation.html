Installation guide

Requirements

What does it require for you to be able to run JSIS?

  • Node.js
  • Node.js modules: ejs, iconv (iconv is actually not strictly necessary)
  • Log files in supported format
  • Some basic skills at running command line tools

How to use it?

  1. Make sure you have installed Node.js and Node Package Manager

    You can probably install them via some installer from the Node.js homepage, or with Node Version Manager. Ask them for more help.

  2. You'll need a place to put the application in, and to run it on .. think of one, and then download the source code from the GitHub project page and put the code there.
  3. Install the node modules "ejs" and "iconv".

    Assuming you've got Node Package Manager installed properly and use BASH, you can open a terminal/whatever to the place you put the files in and run the following:
    modules="ejs iconv"; for module in $modules; do npm install $module; done
    NOTE: iconv most likely needs to be compiled, might have some requirements (such as a compiler), etc. that I cannot, and will not, help you with.

  4. Configure JSIS by copying config.example.js to config.js and editing it to taste. Try to read the documentation if something is unclear.
  5. Run JSIS with:
    node jsis.js
  6. Good luck! If any issues arise, please report them at the GitHub issue tracker ... if you like it, send thanks to Lietu.
